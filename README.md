A cloud-based event management system that allows users to create, view, and register for events, with data stored in AWS RDS (Relational Database Service).

This project demonstrates cloud database integration, where the backend interacts with an RDS database instance hosted on AWS. Amazon RDS provides a managed relational database service for storing application data securely and reliably.
